Sistema de Gestión de Usuarios

Aplicación de consola en Python 3 para administrar usuarios: registra, busca, valida y elimina usuarios, guardando los datos en un archivo de texto con manejo de errores. Incluye control de duplicados, separación de registros válidos/inválidos y fecha de creación.

Ejecutar: python gestion_usuarios.py
